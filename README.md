# タイヤ保管管理システム

ガソリンスタンド向けのタイヤ保管・顧客管理システムです。

## 機能

- 顧客管理（登録、編集、検索）
- ディーラー管理（登録、編集）
- タイヤ管理（登録、保管、引き取り）
- QRコード生成・読み取り
- ロケーション管理（タイヤラック位置管理）
- 保管管理（タイヤ保管状況の可視化）

## 技術スタック

### フロントエンド
- Vue.js 3
- Vue Router
- Axios
- QRコード読み取り・生成ライブラリ

### バックエンド
- laravel
- MySQL
- QRコード生成ライブラリ

### テーブル構成

1. **customers** - 顧客情報
2. **tires** - タイヤ情報
3. **locations** - 保管場所（ラック）情報
4. **tire_storage** - タイヤ保管履歴

## 使用方法

1. 顧客登録
2. タイヤ受け入れ時にQRコードを生成・印刷して貼付
3. 保管場所を選択してロケーション登録
4. 引き取り時にQRコードをスキャンして検索
