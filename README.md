# タイヤ保管管理システム

ガソリンスタンド向けのタイヤ保管・顧客管理システムです。

## 機能

- 顧客管理（登録、編集、検索）
- ディーラー管理（登録、編集）
- タイヤ管理（登録、保管、引き取り）
- QRコード生成・読み取り
- ロケーション管理（タイヤラック位置管理）
- 保管管理（タイヤ保管状況の可視化）

## 技術スタック

### フロントエンド
- Vue.js 3
- Vue Router
- Axios
- QRコード読み取り・生成ライブラリ

### バックエンド
- laravel
- MySQL
- QRコード生成ライブラリ

### テーブル構成

1. Store（店舗）
複数の店舗を管理するためのマスターテーブル
各店舗に固有の店舗コード（store_code）を持つ
店舗ごとに顧客を管理
2. Customer（顧客）
店舗に紐づく顧客情報を管理
顧客コード、連絡先、住所、車両情報を保持
各顧客は複数のタイヤセットを所有可能
3. TireSet（タイヤセット）
顧客が所有するタイヤの情報
QRコードで一意に識別
タイヤのタイプ（夏/冬/オールシーズン）、メーカー、サイズ、状態などを記録
4. TireStorage（タイヤ保管履歴）
タイヤセットの保管・出庫履歴を管理
保管場所（Location）との紐付け
入出庫日、料金、支払状況、担当者を記録
ステータス管理（保管中/出庫済み/予約済み）
5. Location（保管場所）
タイヤの物理的な保管位置を管理
ラック名、セクション、行、列で位置を特定
収容数と利用可否を管理
6. TireImage（タイヤ画像）
タイヤセットに関連する画像ファイルを管理
タイヤの状態確認用の写真を保存
表示順序を管理可能
7. User（ユーザー）

## 使用方法

1. 顧客登録
2. タイヤ受け入れ時にQRコードを生成・印刷して貼付
3. 保管場所を選択してロケーション登録
4. 引き取り時にQRコードをスキャンして検索
